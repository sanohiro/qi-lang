# Qiチュートリアル

**ゼロから学ぶQi言語 - 実践的なコード例で身につける**

このチュートリアルは、Qiの主要機能を段階的に学べるように構成されています。各章は独立しているので、興味のある部分から始めることもできます。

---

## 📚 チュートリアル一覧

### 1. [はじめの一歩](01-getting-started.md) ⭐ 必読
- インストールと環境構築
- 最初のプログラムを実行
- REPLの使い方
- 基本的なデータ型と演算

**所要時間**: 15分

---

### 2. [パイプラインで考える](02-pipelines.md) ⭐ Qiの真髄
- パイプライン演算子 (`|>`)
- データの流れを設計する
- 関数合成とチェーン
- 実用例: テキスト処理、データ変換

**所要時間**: 20分

---

### 3. [パターンマッチングをマスターする](03-pattern-matching.md)
- `match`式の基本
- データ構造の分解
- ガード条件
- 実用例: コマンド処理、メッセージハンドリング

**所要時間**: 25分

---

### 4. [エラーを優雅に扱う](04-error-handling.md) ⭐ 重要
- Railway Pipeline (`|>?`)
- `try`と`error?`述語
- エラー伝播の設計
- 実用例: ファイル処理、API呼び出し

**所要時間**: 30分

---

### 5. [並行・並列処理を簡単に](05-concurrency.md) ⭐ Qiの強み
- 並列パイプライン (`||>`)
- goroutine風の並行処理
- チャネル通信
- 実用例: 並列ダウンロード、データ処理

**所要時間**: 35分

---

### 6. [WebアプリケーションとAPI](06-web-api.md)
- HTTPサーバーの構築
- JSON API
- ルーティングとミドルウェア
- 実用例: RESTful API、CRUD操作

**所要時間**: 40分

---

## 💡 学習のコツ

1. **REPLで試す**: 全てのコード例はREPLで試すことができます
   ```bash
   qi  # REPLを起動
   ```

2. **コードを変えてみる**: 例をコピペするだけでなく、数値やデータを変えて実験しましょう

3. **エラーを恐れない**: エラーメッセージは日本語で表示されます（`QI_LANG=ja`）

4. **小さく始める**: 最初から複雑なコードを書こうとせず、小さな関数から始めましょう
