# Qi Language VSCode Extension

Provides syntax highlighting for the Qi programming language.

## Features
- Syntax coloring for keywords, functions, modules, and operators
- Comment styles (`;;` line, `#| |#` block)
- String and number highlighting
- Special pipeline operators `|>`, `||>`, `|>?`, `~>`, `tap>` highlighted

## Installation
1. Clone or download this repository
2. Open in VSCode
3. Press `F5` to launch Extension Development Host
4. Open a `.qi` file to see highlighting

## Language Configuration
Supports:
- Auto-closing brackets () [] {}
- String closing
- Comment toggling
