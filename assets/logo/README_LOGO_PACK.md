
# Qi Logo Pack (Otter Edition)

This pack contains **editable SVG** logos and **ready-to-use PNG/ICO** assets
based on the official Qi otter mark.

## Files

- `qi-logo-full.svg` — Full lockup (otter + "Qi" + flowing tail) for README titles, hero banners.
- `qi-logo-horizontal.svg` — Compact horizontal layout for website headers.
- `qi-icon.svg` — Icon-only version (circular background) for avatars & app icons.
- `qi-mark.svg` — Minimal mark (flowing tail like `|>`).
- `qi-light.svg` / `qi-dark.svg` — Light/Dark variants for previews and dark-mode UIs.

### PNG / ICO Quick Exports
- `qi-logo-full-[1024|512|256|128|64|48|32].png` — Ready-to-use raster exports.
- `favicon.ico` — Standard multi-size favicon.

> Colors
> - Indigo: #1B2B34
> - Gold:   #E3B23C
> - Brown:  #C6A27C
> - Cream:  #F2E3CF

## Tips
- Prefer **SVG** for docs and websites (it scales perfectly).
- Use `qi-icon.svg` for social avatars or favicon (convert to PNG as needed).
- The tail shape doubles as a nod to the Qi pipeline operator `|>`.
